🤠 One-time setup

Connect a free Firebase database so your whole group's votes sync in real time. Only one person needs to do this — then share the HTML file with the config already saved.

1 Create a Firebase project
Go to console.firebase.google.com → Add project → name it anything → finish the wizard.
2 Enable Realtime Database
Left sidebar → Build → Realtime Database → Create database → pick any location → choose Start in test mode → Enable.
3 Get your config
Gear icon → Project settings → scroll to "Your apps" → click </> → register app → copy the firebaseConfig object.
